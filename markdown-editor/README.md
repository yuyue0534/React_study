# ◈ MdEditor — 在线 Markdown 编辑器

> 基于 **Abricotine** 设计理念构建的前端 Markdown 编辑管理系统

---

## 🚀 快速启动

```bash
# 1. 解压项目后进入目录
cd markdown-editor

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 4. 浏览器自动打开 http://localhost:3000
```

## 📋 功能特性总览

### 📝 核心编辑与预览

| 功能 | 说明 | 快捷键 |
|------|------|--------|
| 实时预览 | 编辑即时渲染，左右分屏 | — |
| 三种视图 | 编辑 / 分屏 / 预览 模式切换 | — |
| 语法高亮 | 代码块自动高亮显示 | — |
| 禅模式 | 全屏无干扰写作 | `F11` |
| 数学公式 | 支持 `$行内$` 和 `$$块级$$` 公式 | — |
| 待办清单 | `- [ ]` / `- [x]` 可视化渲染 | — |

### 🛠️ 文档管理

| 功能 | 说明 | 快捷键 |
|------|------|--------|
| 自动保存 | 编辑后自动保存状态提示 | — |
| 手动保存 | 触发保存状态 | `Ctrl+S` |
| 搜索替换 | 全文搜索与批量替换 | `Ctrl+F` |
| 撤销/重做 | 多步撤销重做历史 | `Ctrl+Z` / `Ctrl+Y` |
| 表格生成 | 可视化配置行列数 | 工具栏按钮 |

### 🎨 自定义外观

| 功能 | 说明 |
|------|------|
| 日间主题 | 暖白色调，适合白天使用 |
| 夜间主题 | 深色背景，适合夜间编辑 |
| 护眼主题 | 米黄色调，长时间阅读友好 |

### 📤 导出功能

| 格式 | 说明 |
|------|------|
| HTML | 完整 HTML 文件含内联样式 |
| Markdown | 导出 .md 源文件 |
| 剪贴板 | 复制渲染后 HTML 到剪贴板 |

### 📊 导航与统计

| 功能 | 说明 |
|------|------|
| 目录侧栏 | 自动提取标题生成导航树 |
| 点击跳转 | 点击目录项滚动到对应位置 |
| 实时统计 | 字数、字符数、行数实时显示 |
| 光标位置 | 状态栏显示当前行列号 |

## 🎹 快捷键一览

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + B` | 粗体 |
| `Ctrl + I` | 斜体 |
| `Ctrl + S` | 保存 |
| `Ctrl + F` | 搜索替换 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Shift + Z` / `Ctrl + Y` | 重做 |
| `F11` | 禅模式 |
| `ESC` | 退出弹窗/禅模式 |

## 🏗️ 项目结构

```
markdown-editor/
├── index.html          # 入口 HTML
├── package.json        # 依赖配置
├── vite.config.js      # Vite 构建配置
├── README.md           # 说明文档
└── src/
    ├── main.jsx        # React 挂载入口
    └── App.jsx         # 编辑器主组件（含全部功能）
```

## 🔧 技术栈

- **React 18** — 前端框架
- **Vite 6** — 构建工具
- **Lodash** — 工具函数（防抖等）
- **自研 Markdown 解析器** — 纯正则实现，零外部依赖

## 📖 支持的 Markdown 语法

- 标题（H1-H6）
- **粗体**、*斜体*、~~删除线~~
- 链接、图片（含标题）
- 行内代码、代码块（含语言标签）
- 有序列表、无序列表
- 待办事项 `- [x]` / `- [ ]`
- 表格（含对齐方式）
- 引用块
- 分割线
- 行内/块级数学公式
- GFM（GitHub Flavored Markdown）
